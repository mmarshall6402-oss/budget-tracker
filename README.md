# Budget Tracker 💰

A modern personal budget tracking web application built with **Python** and **Flask**.  
Track your **income**, **expenses**, **weekly salary**, and **current balance** in a clean, interactive interface with a sleek green & black theme.

---

## Features

- Set a **starting balance**
- Add **income** or **expenses** with categories
- Add **weekly salary** by hours worked and hourly rate
- **Clear all data** with a single button
- Modern UI with **dark mode** and interactive buttons
- Displays **current balance** and categorized summaries
- Built using **Python, Flask, HTML, and CSS**
